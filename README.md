# GitPro

[Check it out here](https://gitpro.app)

**Install Dependencies**
```sh
npm install
```

**Start the Server**
```sh
npm start
```

**Start the Server(in Development Mode)**
```sh
npm run dev
```

**Start Worker**
```sh
npm run task
```

**Start Worker(in Development Mode)**
```sh
npm run task-dev
```
